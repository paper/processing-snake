# processing-snake
snake game by processing

![](./images/game.jpg)

### 缘由

几年前写过DOM版本的贪吃蛇，这次使用 [processing](https://processing.org/) 来写，
又是另外一种感觉。很棒，很不错！

再加上 [processing-js](https://github.com/jeresig/processing-js) 的支持，使得pde文件可以运行在web上，编译成 js 代码在 canvas 上运行。
